This project was made at random, mainly because i had nothing else to do 😅
It started when i was going through TikTok, like many people, and i stumbled accross a video explaining how the ID in TikTok videos can be used to find the Exact date the Video was uploaded

So, using the idea from this TikTok, Hes credited ofc, I have created a website that allows users to enter a Link to get the Date of the TikTok Video.

Long story short, Take the ID, Convert to Binary, Take the first 32 Didgits, Convert to Denary then finally convert using the UNIX timestamp converter, I made my own.

Hope you link this mini Web App of mine 😎
